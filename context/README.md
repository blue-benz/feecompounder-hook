# YieldStream Context

This folder collects local reference material for the YieldStream Hook build.

## UHI Workshops

- `uhi-workshops/workshops` - Atrium Academy workshop examples for Uniswap Hook Incubator.

## Uniswap

- `uniswap-docs/docs` - Uniswap documentation.

## Reactive Network

Official links:

- Docs: https://dev.reactive.network/
- Education course: https://dev.reactive.network/education/introduction
- Smart contract demos: https://github.com/Reactive-Network/reactive-smart-contract-demos
- Developer Telegram: https://t.me/reactivedevs

Most relevant local folders:

- `reactive-network/documentation` - Reactive docs and education archive.
- `reactive-network/reactive-smart-contract-demos` - RSC examples.
- `reactive-network/reactive-hardhat-demos` - Hardhat demos.
- `reactive-network/hackathon` - Hackathon boilerplate.
- `reactive-network/reactive-lib` - Reactive base contracts.
- `reactive-network/reactive-lib-omni` - Omni Reactive base contracts.
- `reactive-network/reactive-test-lib` - Foundry simulator for RSC tests.
- `reactive-network/system-smart-contracts` - Subscription/callback system contracts.

These folders are nested Git checkouts for reference and are not root-level submodules.
